DiviTheatre Motion

DiviTheatre is a free WordPress plugin that adds cinematic scroll animations to Divi 5 — fade-up, stagger, parallax, hero-reveal, and more. It runs on Theatre.js and requires no page builders or shortcodes.

Animations it adds:

  • fade-up — text and sections fade in as they scroll into view
  • stagger — card grids animate in sequence, 100ms apart
  • parallax-scroll — hero images drift as you scroll
  • hero-reveal — choreographed hero: background → headline → body → CTA
  • hover-grow — buttons and cards gently scale on hover
  • scale-in — images and blurbs zoom in from 90%

How to install:

  1. Download the plugin ZIP from your iConnectIT account
  2. WordPress admin → Plugins → Add New → Upload Plugin
  3. Upload the ZIP and click Install Now, then Activate
  4. No configuration needed — it activates automatically on Divi 5 pages

If you select "Not installed but I want it", the generator builds a static page and notes exactly which elements should get animations once you install it.

Install the Import Plugin

The Divi Tools Importer plugin lets you push generated JSON pages directly into any WordPress site — no FTP or WP-CLI needed.

Step 1 — Download the plugin ZIP

The plugin is bundled with this app. Click below to download it:

Download divi-tools-importer.zip

Step 2 — Install in WordPress

  1. Go to WordPress admin → Plugins → Add New Plugin
  2. Click Upload Plugin
  3. Choose the downloaded ZIP and click Install Now
  4. Click Activate Plugin

Step 3 — Get your API key

  1. Go to WordPress admin → Settings → Divi Tools Importer
  2. Copy the Site URL and API Key shown there
  3. Paste both into the fields above and click Save

What it does: After generating a page, the app sends the JSON to your site via the plugin's REST API. The page is created as a draft — you preview it in Divi, then publish when happy. No manual file copying needed.

Works with any WordPress host — Local, Kinsta, WP Engine, SiteGround, or any other. The site just needs to be reachable from this Mac.
