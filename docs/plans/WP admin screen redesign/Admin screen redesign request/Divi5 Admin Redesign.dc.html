Installing the Pro skills
  1. Upgrade, then download the toolkit from your iConnectIT account — you get a licensed .zip (it isn't on the public marketplace).
  2. Unzip it, then point Claude Code at the folder:
$ claude plugin marketplace add ~/Downloads/divi5generate
$ claude plugin install divi5generate@divi5generate

Unlocks the advanced skills — full page generator, brand systems, deploy & QA — plus every Pro connector feature on this site.
